Notable additions and changes to the Actioneer documentation and integrations, newest first.

<Update label="June 2026" tags={["Docs"]}>
  ## Get Started, Concepts, and Resources

  Introduced a documentation home with a [Quickstart](/quickstart), a [Concepts](/concepts/how-actioneer-reads-your-data) layer, and a Resources section covering [Support & Security](/support), an [FAQ](/faq), and [Access & networking](/operations/access-and-networking).
</Update>

<Update label="June 2026" tags={["Ingestion"]}>
  ## CDP Event Ingestion API

  Published the [Event Ingestion](/ingestion) section: the `/capture` and `/batch` endpoints with an interactive API reference, the [identity model](/ingestion/identity), [property taxonomy](/ingestion/properties), [reliability semantics](/ingestion/reliability), and an [errors reference](/ingestion/errors).
</Update>

<Update label="June 2026" tags={["Integrations"]}>
  ## Data source & MMP integrations

  Published connection guides for warehouses and query engines (Snowflake, BigQuery, Redshift, Athena, Databricks, ClickHouse, PostgreSQL), object storage and files (Amazon S3, Google Cloud Storage, Google Sheets), and Mobile Measurement Partners (Adjust, AppsFlyer, Singular).
</Update>
