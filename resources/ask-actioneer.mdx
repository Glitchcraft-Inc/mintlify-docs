These docs are built to be read by people **and** by AI agents. You can search and ask questions inside the docs, or connect them to your own tools over MCP.

## Ask inside the docs

<CardGroup cols={2}>
  <Card title="Search" icon="search">
    Press <kbd>Cmd</kbd>/<kbd>Ctrl</kbd> + <kbd>K</kbd> anywhere to jump to any page.
  </Card>
  <Card title="Ask AI" icon="sparkles">
    Use **Ask AI** to get an answer synthesized from these docs, with links back to the source pages.
  </Card>
</CardGroup>

## Connect over MCP

Add the Actioneer docs as an MCP server so your IDE or coding agent can read them directly while you build an integration. Use the **MCP** option in the page menu (top of any page) to copy the server details, or **Add to MCP** to set it up in a supported client.

<Info>
  Connecting the docs MCP server is read-only — it exposes documentation to your agent. It does not grant any access to your Actioneer workspace or data.
</Info>

## Machine-readable exports

Every page is available as plain Markdown for tools and agents, and the full site is published as a single text file:

- `/llms.txt` — an index of the documentation for LLMs.
- `/llms-full.txt` — the entire documentation as one text file.
