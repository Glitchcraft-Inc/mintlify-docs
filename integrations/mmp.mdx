Attribution data, campaign performance, install events — **pulled into the same context layer as your warehouse and BI tools.**

Actioneer connects to your Mobile Measurement Partner via its reporting API, giving you a single place to query installs, in-app events, campaign spend, and attribution data alongside everything else in your stack. Each connection is read-only and takes about 10 minutes to set up.

## Supported partners

<CardGroup cols={3}>
  <Card title="Adjust" icon="smartphone" href="/integrations/mmp/adjust">
    Query attribution data, campaign metrics, and install events from Adjust.
  </Card>
  <Card title="AppsFlyer" icon="smartphone" href="/integrations/mmp/appsflyer">
    Query attribution data and campaign performance from AppsFlyer. Requires Master API.
  </Card>
  <Card title="Singular" icon="smartphone" href="/integrations/mmp/singular">
    Query attribution data, campaign metrics, and install events from Singular.
  </Card>
</CardGroup>

<Tip>
  All MMP connections use read-only API credentials (tokens or keys, depending on the platform). Actioneer never modifies your MMP configuration, postbacks, or attribution settings.
</Tip>

<Info>
  After connecting, you choose which apps Actioneer can query. You can update your selection at any time from the connection settings.
</Info>
