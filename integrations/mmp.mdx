# MMP Connections

Attribution data, campaign performance, install events — **pulled into the same context layer as your warehouse and BI tools.**

Each guide covers prerequisites, permissions, and the exact fields you need to complete the connection.

<CardGroup cols={3}>
  <Card title="Adjust" href="/integrations/mmp/adjust">
    Query attribution data, campaign metrics, and install events from Adjust.
  </Card>
  <Card title="AppsFlyer" href="/integrations/mmp/appsflyer">
    Query attribution data and campaign performance from AppsFlyer. Requires Master API.
  </Card>
  <Card title="Singular" href="/integrations/mmp/singular">
    Query attribution data, campaign metrics, and install events from Singular.
  </Card>
</CardGroup>

<Info>
  Actioneer connects to MMP platforms in read-only mode. It can query attribution data, campaign performance, and install events but cannot modify any configuration, postbacks, or attribution settings.
</Info>
