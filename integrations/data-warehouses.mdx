# Integration Guides

Each guide covers prerequisites, permissions, and the exact fields you need to complete the connection in Actioneer.

<CardGroup cols={2}>
  <Card title="Amazon Redshift" icon="warehouse" href="/integrations/data-warehouses/amazon-redshift">

  <Card title="Amazon S3" icon="archive" href="/integrations/data-warehouses/s3">

  </Card>
  <Card title="Databricks" icon="blocks" href="/integrations/data-warehouses/databricks">

  </Card>

<Info>
  Actioneer connects in read-only mode across all integrations. It cannot write, delete, or modify any data in your warehouse.
</Info>
