Your data source is the foundation. **Connect it once — Actioneer queries it in place, no data migration.**

Each guide covers prerequisites, permissions, and the exact fields you need to complete the connection.

<Tip>
  All connections use **least-privilege permissions** scoped to query execution and data reads. Actioneer's IP for whitelisting: `35.244.14.238`.
</Tip>

<Info>
  Actioneer connects with the minimum permissions required for each source. It cannot create, modify, or delete any tables, schemas, or source data. Where a service requires write access for its own operations (e.g., Athena writes temporary query results to S3), permissions are scoped to that specific purpose only.
</Info>
