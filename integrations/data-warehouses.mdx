# Data Warehouses

Your warehouse is the foundation. **Connect it once — Actioneer queries it in place, read-only, no data migration.**

Each guide covers prerequisites, permissions, and the exact fields you need to complete the connection.

<Tip>
  All warehouse connections are **read-only** (`SELECT` only). Actioneer's IP for whitelisting: `35.244.14.238`.
</Tip>

  <Card title="Amazon Redshift" icon="/images/amazon-redshift.svg" href="/integrations/data-warehouses/amazon-redshift">

  <Card title="Google BigQuery" icon="/images/google-big-query.svg" href="/integrations/data-warehouses/google-big-query">

  </Card>

<Info>
  Actioneer connects in read-only mode across all integrations. It cannot write, delete, or modify any data in your warehouse.
</Info>
