# Campaigns

Campaigns in Actioneer connect the analysis layer to the action layer. When you identify a cohort worth reaching — through Chat, Funnels, Retention, or Segments — Campaigns let you push that audience to your CLM or messaging platform without leaving Actioneer.

---

## How Campaigns Work

| Step | Description |
| :--- | :--- |
| **Define the audience** | Select an existing Segment or build one on the fly using any combination of properties, behaviours, and event sequences |
| **Choose the destination** | Select the CLM or messaging platform — CleverTap, MoEngage, WebEngage, Braze, or others |
| **Push** | Actioneer sends the Segment to the destination tool. Your messaging platform handles delivery. |

---

## Scope

| Campaigns do | Campaigns do not |
| :--- | :--- |
| Push audience lists to external CLM tools | Compose or send messages directly |
| Enable cohort-to-campaign workflows in minutes | Replace your existing messaging platform |
| Connect insight to action without tool-switching | Manage notification templates or delivery schedules |

<Info>
  Actioneer handles audience definition and push. Message composition, A/B testing, and delivery scheduling remain in your CLM tool — where those features are purpose-built.
</Info>

---

## Campaign Triggers

| Trigger | Description |
| :--- | :--- |
| **Manual** | Build a Segment, select a destination, and push. One-time execution. |
| **From a Playbook** | A Playbook identifies an anomaly, builds a Segment from the affected users, and pushes it to your CLM. |
| **From a Scout alert** | A Scout detects a KPI anomaly and triggers a pre-configured Campaign response. |

---

## Use Cases

| Scenario | How it works |
| :--- | :--- |
| **Cart recovery** | Identify users who added items but did not complete purchase in the last 24 hours. Push to your CLM for a nudge. |
| **Re-engagement** | Find KYC-complete users with zero investment and 30+ days of dormancy. Push to a winback campaign. |
| **Cross-sell** | Surface users with high activity in one product category but zero engagement in another. Target for a cross-sell flow. |
| **Regional response** | Detect a retention drop in a specific city. Build a targeted Segment and push a localised campaign. |
