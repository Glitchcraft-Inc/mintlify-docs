# Boards

Every chart your team keeps going back to — **in one place, always current.**

Boards are persistent, organised pages where findings from conversations, Playbooks, and other analyses are saved, revisited, and compared. Think of them as dashboards built through conversation — not through a drag-and-drop editor.

---

## Creating a Board

| &nbsp; Method | &nbsp; How to do it | &nbsp; Example prompt |
| :--- | :--- | :--- |
| &nbsp; **New board** | &nbsp; Ask Actioneer to create one | &nbsp; *"Create a new board for weekly performance tracking"* |
| &nbsp; **Customised board** | &nbsp; Specify exactly what goes on it — metrics, chart types, filters, layout | &nbsp; *"Build a board with DAU trend, revenue breakdown by country, and top 10 campaigns by spend"* |
| &nbsp; **Template-based** | &nbsp; Start from a pre-built template | &nbsp; *"Create a board using the retention template"* |
| &nbsp; **From an existing deck** | &nbsp; Reference a previous analysis or playbook | &nbsp; *"Turn my last retention analysis into a board"* or *"Create a board from the Q1 performance deck"* |
| &nbsp; **By pinning** | &nbsp; Pin any result from conversations, Funnels, Retention, or Segments | &nbsp; One-click pin from any analysis |

**Common templates:**

| &nbsp; Template | &nbsp; What's included |
| :--- | :--- |
| &nbsp; **Executive Summary** | &nbsp; Top KPIs at a glance |
| &nbsp; **UA Performance** | &nbsp; CPI, installs, spend by channel |
| &nbsp; **Engagement** | &nbsp; DAU, sessions, retention curves |
| &nbsp; **Monetisation** | &nbsp; Revenue, ARPU, conversion rates |

### Customising a board

When building a customised board, you can specify:
- **Metrics** — choose from existing metrics defined on the Metrics page
- **Visualisations** — line charts, bar charts, tables, KPI cards
- **Filters** — date ranges, segments, platforms, regions
- **Layout** — group related metrics with titles and sections

Each element on the board is independently configurable.

---

## Board Structure

| &nbsp; Element | &nbsp; Description |
| :--- | :--- |
| &nbsp; **Section** | &nbsp; A named group of related cards. Collapsible, reorderable, and renamable. |
| &nbsp; **Card** | &nbsp; A single visualisation: line chart, bar chart, table, or metric number. |
| &nbsp; **Insight line** | &nbsp; A short annotation attached to a section summarising the takeaway. |

Actioneer auto-detects the appropriate visualisation:

| &nbsp; Result Shape | &nbsp; Default Card Type |
| :--- | :--- |
| &nbsp; Time series | &nbsp; Line chart |
| &nbsp; Single number | &nbsp; Metric card |
| &nbsp; Ranked list | &nbsp; Bar chart |
| &nbsp; Tabular data | &nbsp; Table |

## Managing Boards

| &nbsp; Action | &nbsp; How to do it | &nbsp; Example prompt |
| :--- | :--- | :--- |
| &nbsp; **Rename** | &nbsp; Rename a Board, section, or card at any time | &nbsp; — |
| &nbsp; **Reorder** | &nbsp; Drag sections into place | &nbsp; — |
| &nbsp; **Global date filter** | &nbsp; Update all time-based cards at once | &nbsp; — |
| &nbsp; **Delete** | &nbsp; Remove a Board permanently — confirmation required | &nbsp; *"Delete the weekly performance board"* |

<Info>
  Boards are scoped to the active dataset. Deleting a Board does not affect underlying data or metrics.
</Info>
