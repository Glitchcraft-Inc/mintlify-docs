# Boards

Every chart your team keeps going back to — **in one place, always current.**

Boards are persistent, organised pages where findings from conversations, Playbooks, and other analyses are saved, revisited, and compared. Think of them as dashboards built through conversation — not through a drag-and-drop editor.

---

## Creating a Board

| Method | How to do it | Example prompt |
| :--- | :--- | :--- |
| **New board** | Ask Actioneer to create one | *"Create a new board for weekly performance tracking"* |
| **Customised board** | Specify exactly what goes on it — metrics, chart types, filters, layout | *"Build a board with DAU trend, revenue breakdown by country, and top 10 campaigns by spend"* |
| **Template-based** | Start from a pre-built template | *"Create a board using the retention template"* |
| **From an existing deck** | Reference a previous analysis or playbook | *"Turn my last retention analysis into a board"* or *"Create a board from the Q1 performance deck"* |
| **By pinning** | Pin any result from conversations, Funnels, Retention, or Segments | One-click pin from any analysis |

**Common templates:**

| Template | What's included |
| :--- | :--- |
| **Executive Summary** | Top KPIs at a glance |
| **UA Performance** | CPI, installs, spend by channel |
| **Engagement** | DAU, sessions, retention curves |
| **Monetisation** | Revenue, ARPU, conversion rates |

### Customising a board

When building a customised board, you can specify:
- **Metrics** — choose from existing metrics or create new ones
- **Visualisations** — line charts, bar charts, tables, KPI cards
- **Filters** — date ranges, segments, platforms, regions
- **Layout** — group related metrics with titles and sections

Each element on the board is independently configurable.

---

## Board Structure

| Element | Description |
| :--- | :--- |
| **Section** | A named group of related cards. Collapsible, reorderable, and renamable. |
| **Card** | A single visualisation: line chart, bar chart, table, or metric number. |
| **Insight line** | A short annotation attached to a section summarising the takeaway. |

Actioneer auto-detects the appropriate visualisation:

| Result Shape | Default Card Type |
| :--- | :--- |
| Time series | Line chart |
| Single number | Metric card |
| Ranked list | Bar chart |
| Tabular data | Table |

## Managing Boards

| Action | How to do it | Example prompt |
| :--- | :--- | :--- |
| **Rename** | Rename a Board, section, or card at any time | — |
| **Reorder** | Drag sections into place | — |
| **Global date filter** | Update all time-based cards at once | — |
| **Delete** | Remove a Board permanently — confirmation required | *"Delete the weekly performance board"* |

<Info>
  Boards are scoped to the active dataset. Deleting a Board does not affect underlying data or metrics.
</Info>
