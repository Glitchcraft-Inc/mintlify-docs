# Chat

Chat is the primary interface for interacting with Actioneer. It is a natural-language conversation layer that translates plain-English questions into SQL queries, runs them against your connected data warehouse, and returns structured answers — charts, tables, or single metrics.

---

## How it works

Type a question the way you would ask a colleague. Actioneer handles the rest.

<Steps>
  <Step title="Ask a question">
    Write your question in plain English. No SQL, no query syntax, no specific formatting required.

    Example: *"What was conversion from app download to first investment last week, broken down by city?"*
  </Step>
  <Step title="Actioneer interprets and plans">
    The question is parsed for intent, data scope, and relevant signals. Actioneer maps it to your schema and metric definitions, then generates a query plan.
  </Step>
  <Step title="Queries execute">
    SQL is generated, validated, and run against your warehouse. For complex questions, multiple queries may run in sequence — each building on the results of the previous one.
  </Step>
  <Step title="Answer delivered">
    Results are returned as a chart, table, or metric card. The underlying SQL is always visible alongside the answer.
  </Step>
</Steps>

---

## What you can ask

Chat is not limited to predefined reports. It handles ad-hoc questions across any data your warehouse contains.

| Question type | Example |
| :--- | :--- |
| **Metric lookups** | "What was DAU last week?" |
| **Breakdowns** | "Show me retention by acquisition channel for the March cohort." |
| **Comparisons** | "Compare Android vs iOS funnel conversion for Q1." |
| **Anomaly investigation** | "Why did signups drop on Tuesday?" |
| **Segment exploration** | "How many users completed KYC but never invested?" |
| **Time-series** | "Plot weekly revenue for the last 12 weeks." |

---

## Research mode

For questions that require deeper investigation — spanning multiple tables, intermediate analysis, or conditional logic — Chat enters research mode. Instead of a single query, Actioneer runs a multi-step investigation and returns a structured report with findings, supporting data, and recommendations.

Research mode activates automatically when the question warrants it. You do not need to enable it manually.

<Info>
  Every answer in Chat is traceable. The SQL query, data source, and execution time are displayed alongside the result. Nothing is a black box.
</Info>

---

## From Chat to action

Chat is not a dead end. Any result can flow into other parts of the platform:

- **Pin to a Board** — Save a chart or metric to a persistent dashboard.
- **Save as a Playbook** — Codify a multi-step analysis so it can be rerun with fresh data.
- **Build a Segment** — Turn a cohort from a query result into a targetable audience.
- **Set an alert** — Ask Actioneer to monitor a metric and notify you when it crosses a threshold.
