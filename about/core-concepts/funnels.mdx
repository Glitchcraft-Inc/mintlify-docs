# Funnels

A Funnel is a sequential analysis of user steps — tracking how users move through a defined series of actions and where they drop off. Funnels help identify conversion bottlenecks and optimise user flows.

---

## Creating a Funnel

Ask in plain language or use the Funnels page:
- *"Create a funnel for onboarding: install → tutorial start → tutorial complete → first session"*
- *"Build a purchase funnel from app open to checkout"*

**Required fields:**

| Field | Description | Example |
| :--- | :--- | :--- |
| **Name** | Unique funnel label | `onboarding_funnel` |
| **Steps** | Ordered sequence of events | Step 1: `app_open`, Step 2: `tutorial_start`, Step 3: `first_purchase` |
| **Filters** *(optional)* | Narrow the audience | Platform, country, date range, segment |
| **Time window** *(optional)* | Max time allowed between steps | *"Complete all steps within 7 days"* |
| **Breakdown** *(optional)* | Split results by a dimension | By country, platform, campaign |

Process: You define the steps → Actioneer validates events exist in your data → funnel is computed showing user counts and drop-off % at each step → results are saved for reuse.

---

## Funnel Components

| Component | Description |
| :--- | :--- |
| **Steps** | Ordered sequence of up to 10 events |
| **Conversion rate** | Percentage of users who progressed from one step to the next |
| **Drop-off** | Users who completed step N but did not reach step N+1 |
| **Time between steps** | Median, mean, and P95 duration between consecutive steps |

---

## Analysis Views

| View | Description |
| :--- | :--- |
| **Step-by-step conversion** | Raw count and percentage at each step — identifies the biggest drop-off point |
| **Time-to-convert** | How long users take to move between steps |
| **Segment comparison** | Overlay two or more Segments on the same Funnel |
| **Drop-off diagnosis** | See what events dropped users performed after leaving the funnel |

---

## Managing Funnels

| Action | How to do it | Example prompt |
| :--- | :--- | :--- |
| **Edit** | Add, remove, or reorder steps. Adjust filters or time window. | *"Add 'level 5 reached' after tutorial complete"* or *"Change the time window to 3 days"* |
| **Pin to a Board** | Attach the funnel chart for ongoing monitoring. Pinned funnels refresh automatically. | *"Pin the onboarding funnel to my engagement board"* |
| **Delete** | Permanently remove — confirmation required. Does not affect underlying event data. | *"Delete the onboarding funnel"* |

**Edit types:**

| Edit | Example |
| :--- | :--- |
| Add a step | *"Insert 'add to cart' between browse and checkout"* |
| Remove a step | *"Drop the registration step"* |
| Reorder steps | *"Move 'first purchase' before 'profile setup'"* |
| Change filters | *"Filter to iOS users only"* |
| Adjust time window | *"Allow 14 days between steps instead of 7"* |
| Add breakdown | *"Break down by country"* |

<Info>
  Funnels support up to 10 steps. For longer flows, break them into sub-funnels and analyse independently.
</Info>
