# Funnels

Funnels in Actioneer track user progression through a defined sequence of events — from first touch to conversion, or any multi-step flow your product involves. They surface where users drop off, how long each step takes, and how different Segments compare.

---

## Funnel Components

| Component | Description |
| :--- | :--- |
| **Steps** | Ordered sequence of up to 10 events, e.g. App Open → Sign Up → KYC → First Investment |
| **Conversion rate** | Percentage of users who progressed from one step to the next |
| **Drop-off** | Users who completed step N but did not reach step N+1 |
| **Time between steps** | Median, mean, and P95 duration between consecutive steps |

---

## Creating a Funnel

| Method | Description |
| :--- | :--- |
| **From Chat** | Ask "Show me the funnel from app download to first investment" and Actioneer builds it |
| **From the Funnels page** | Define steps manually by selecting events from your schema |

Each step can include property filters to narrow the scope — e.g. "Sign Up" where platform is Android, or "Purchase" where amount exceeds ₹500.

---

## Analysis Views

| View | Description |
| :--- | :--- |
| **Step-by-step conversion** | Raw count and percentage at each step. Identifies the single biggest drop-off point. |
| **Time-to-convert** | How long it takes users to move between steps. Median and P95 times distinguish healthy progression from stalled users. |
| **Segment comparison** | Overlay two or more Segments on the same Funnel. Compare Android vs iOS, paid vs organic, or any dimension. |
| **Drop-off diagnosis** | For users who dropped at a specific step, see what events they performed after — across sessions. |

---

## Acting on Funnel Insights

| Action | Description |
| :--- | :--- |
| **Save drop-off users as a Segment** | Target users who fell off at a specific step for re-engagement |
| **Pin to a Board** | Add the Funnel chart to a Board for ongoing monitoring |
| **Set an alert** | Monitor a step's conversion rate and get notified when it drops below a threshold |
| **Build a Campaign** | Push the drop-off Segment to your CLM and trigger a nudge |

<Info>
  Funnels support up to 10 steps. For longer flows, break them into sub-funnels and analyse them independently.
</Info>
